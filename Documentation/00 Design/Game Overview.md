
# Game Overview

Sandbox game where players can edit the elemental makeup of any object; such as a dumbell being given helium characteristics, so instead of it being heavy, its really light and floats around, or a steel wall being given wood characteristics so it can break easily or burn.

## Genre
- Sandbox 
- Physics
- Single player
## Perspective
- Floating cam

## Target Playtime
- 20-30 mins

# Gameplay loop
- spawn in room
- find objects
- choose an element
- apply element
- observe physics change
- have fun and muck about

# Design goals
- Changing elements should be easy and intuitive
- wacky nature should be apparent quickly
- physics changes should feel immediate and satisfying 

# Technical goals
- built in unreal engine c++
- no hardcoded behaviors
- gameplay system should be intuitive 
- OOP!

## Core Mechanic
- Change elements of objects

## Goal
- mess around
- solve puzzles
